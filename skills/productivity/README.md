# 生产力

通用工作流工具，不仅限于编码。

- **[caveman](./caveman/SKILL.md)** — 超压缩沟通模式。通过去除冗余内容将 token 消耗降低约 75%，同时保持完整的技术准确性。
- **[grill-me](./grill-me/SKILL.md)** — 持续被追问你的计划或设计，直到决策树上的每个分支都被解决。
- **[handoff](./handoff/SKILL.md)** — 将当前对话压缩为交接文档，以便另一个 agent 可以继续工作。
- **[write-a-skill](./write-a-skill/SKILL.md)** — 创建结构合理、渐进式呈现、包含打包资源的新 skill。
