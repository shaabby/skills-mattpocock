# 开发中

仍在开发中的 Skills。尚未准备好发布——可能会有粗糙的边缘情况、破坏性变更和被放弃的实验。在毕业到稳定分类之前，它们不会出现在插件和顶层 README 中。

- **[review](./review/SKILL.md)** — 从两个并行维度审查自某个固定点以来的变更：**标准**（diff 是否符合仓库的编码标准？）和**规格**（diff 是否忠实地实现了原始 issue/PRD？）。
- **[writing-beats](./writing-beats/SKILL.md)** — 以节拍之旅的方式塑造文章，选择你自己的冒险风格。选择一个起始节拍，只写那个节拍，然后转向下一个，直到文章达到自然的终点。
- **[writing-fragments](./writing-fragments/SKILL.md)** — 深度访谈式对话，从你身上挖掘碎片——各种异质的写作素材——并将其追加到一个文档中，作为未来文章的原材料。
- **[writing-shape](./writing-shape/SKILL.md)** — 将原始素材的 markdown 文件逐段塑造成文章，每一步都论证格式选择。
